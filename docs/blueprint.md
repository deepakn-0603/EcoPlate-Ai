# **App Name**: WeatherVision

## Core Features:

- Live Weather Data: Fetch and display real-time weather data for any location using a weather API.
- Location Search: Allow users to search for weather by city, state, or country with auto-complete suggestions.
- Dynamic Backgrounds: Change the app's background based on the time of day and current weather conditions.
- Animated Weather Icons: Display animated icons that reflect current weather conditions to enhance the user interface.
- Temperature Toggle: Allow users to toggle between Celsius and Fahrenheit for temperature displays.
- Hourly Weather tool: Use generative AI to analyze historical weather data and refine weather API data with relevant data to predict the weather on an hourly basis. Present a list of likely weather conditions for the current hour

## Style Guidelines:

- Primary color: A deep sky blue (#49A8FF) to evoke a sense of clarity and openness, fitting for a weather app.
- Background color: A very light, desaturated blue (#E5F3FF) to provide a clean and unobtrusive backdrop, brightness adjusted for a light color scheme.
- Accent color: A slightly greenish blue (#49FFB3) to complement the primary and offer contrast for interactive elements and highlights.
- Headline font: 'Space Grotesk', a modern sans-serif for a clean and techy feel; body font: 'Inter', a neutral sans-serif, for readability
- Use custom-designed, animated weather icons to represent different conditions (sunny, rainy, snowy, etc.).
- Implement a dashboard layout with a hero section for main weather info and detailed metrics in cards.
- Employ smooth transitions and micro-interactions throughout the app to enhance user experience.
